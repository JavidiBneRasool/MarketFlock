# 📘 Quick Link: Facebook Graph API Explorer

To get the **Page Token** we need, please go directly to this tool:

👉 **[Graph API Explorer](https://developers.facebook.com/tools/explorer/)**

### Once there, follow these 3 clicks:
1.  **Meta App**: Select `AutoFlock`.
2.  **User or Page**: Click the dropdown and choose **"Get Page Access Token"**. Select your Page.
3.  **Generate**: Click the blue **Generate Access Token** button.

Copy that long string and paste it here!
